# Kredit Ikon Animasi

Aplikasi memakai ikon animasi dari **Flaticon** (https://www.flaticon.com/animated-icons-most-downloaded).
Paket gratis Flaticon **mewajibkan atribusi**. Catat setiap ikon yang dipakai di sini:

| Ikon (file) | Judul | Pembuat | URL sumber |
|-------------|-------|---------|------------|
| _contoh_ parking.json | Parking | Freepik | https://www.flaticon.com/... |

> Saat ini UI memakai emoji + animasi CSS sebagai placeholder. Untuk mengganti dengan
> Lottie Flaticon: simpan file `.json` di folder ini, lalu render via lottie-web
> (mis. `https://cdnjs.cloudflare.com/ajax/libs/lottie-web/5.12.2/lottie.min.js`)
> pada slot ikon (lihat `js/parts.js` dan komponen `.tile .ic`).

Contoh atribusi yang harus tampil di app (mis. di halaman Akun/About):
`Animated icons created by Freepik - Flaticon`
